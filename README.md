# SMARC15
Saint-Mamet Auto Radio Club, official web site by goodup302 with love &lt;3.


## Objectives
Development and deployment of the communication website for the SMARC RC club. The main objective of the client is to reinforce it's precence online by using the visibility of their own website to promote their various events and infrastructures to attract more enthousiast about the hobby.

## Project requirements
Here is an overview of the first requirements expressed by the client for the V1 of the website :

Front-end :
- Most of the content will be displayed on main home page
- Simple layout with header and footer that uses anchors to navigate through the page.
- Sliders (slicks) of high definitions photos *use placeholders  at first*
- Different paragraphs to describe the club (events, history, photos descriptions etc...) *use lorems at first*
- Club Contact informations *will be defined later and communicated privately*
- Club localisation information (map point) *or if not possible static sattelite view image + address*
- Presentation video on a default media player *use placeholder at first, will be communicated later*
- FFRC federation mentions + informations (logo, info, etc...) *will be defined later*
- League AURA mentions *to be defined with client*
- Contact form or contact informations *will be communicated privately*
- Implementation of facebook feed (preferable option) or link to socials (facebook)
- 

Back-end :
- Administration interface (Wordpress / Gutenberg)
- SEO coverage
- Traffic monitoring
- Contact form
